```
    设备层
    （485设备 / CAN设备）
            │
            ▼
    协议转换层
    （485 → TCP 网关 / CAN → TCP 网关）
            │
            ▼
    EMS网关（数据汇聚）
            │
            ▼
    ThingsBoard（数据接入 / 处理 / 存储）
            │
            ▼
    后台系统（展示 / 业务处理）
```

## thingsboard存储的字段
```
| 字段名       | 类型      | 含义           | 实际作用               | 典型内容                    |
| --------- | ------- | ------------ | ------------------ | ----------------------- |
| entity_id | uuid    | 设备ID         | 标识数据属于哪个设备         | device.id               |
| key       | integer | 指标类型ID（字典编码） | 标识“是什么数据”          | voltage=33 / current=42 |
| ts        | bigint  | 时间戳（毫秒）      | 数据产生时间 + 分区依据      | 1775001600000           |
| bool_v    | boolean | 布尔值          | 存储 true/false 类型数据 | 开关状态                    |
| str_v     | varchar | 字符串值         | 存储文本/枚举状态          | ENABLED / OFF           |
| long_v    | bigint  | 长整型数值        | 存储整数型指标            | 计数器 / 累计值               |
| dbl_v     | double  | 浮点数值         | 存储实时传感器数据（最常用）     | 电压 220.5 / 电流 10.2      |
| json_v    | json    | JSON对象       | 存储复杂结构数据           | 多相电压 / 组合数据             |

```

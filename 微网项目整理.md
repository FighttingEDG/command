# windows的各服务操作
## zentao
- 超级管理：sudo -i
- 重启：/opt/zbox/zbox restart
## nacos
- nacos启动：cd D:\nacos\bin；startup.cmd -m standalone

## kafka
- java启动(目前手动启动，后续用nssm来做注册进服务)
    - cd D:\kafka_2.13-4.2.0
    - .\bin\windows\kafka-run-class.bat kafka.Kafka config\server.properties
- 创建topic：
    ```
    .\bin\windows\kafka-topics.bat ^
    --bootstrap-server localhost:9092 ^
    --create ^
    --topic exchange-rate ^
    --partitions 1 ^
    --replication-factor 1
    ```
- 查看当前的topic：.\bin\windows\kafka-topics.bat --bootstrap-server localhost:9092 --list
- 发送消息(生产者)：.\bin\windows\kafka-console-producer.bat --topic test --bootstrap-server localhost:9092
- 接收消息(消费者)：
    ```
    .\bin\windows\kafka-console-consumer.bat ^
    --topic test ^
    --from-beginning ^
    --bootstrap-server localhost:9092
    ```
